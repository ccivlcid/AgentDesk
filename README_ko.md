# AgentDesk

AI 에이전트 오피스 시뮬레이터 - CEO 데스크에서 AI 에이전트 팀을 지휘하세요.

## 주요 기능

- 픽셀아트 애니메이션 오피스 (부서별 에이전트 이동)
- CEO 대시보드 (KPI 메트릭, 에이전트 랭킹)
- 다중 AI 프로바이더 지원 (Claude Code, Codex CLI, Gemini CLI 등)
- 칸반 보드 기반 태스크 관리
- 메신저 연동 (Telegram, Discord, Slack 등)
- 다국어 UI (영어, 한국어, 일본어, 중국어)

## 요구사항

- Node.js >= 22
- pnpm >= 10

## 빠른 시작

```bash
pnpm install
cp .env.example .env
pnpm setup
pnpm dev
```

브라우저에서 http://localhost:8800 을 열어주세요.
