## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| **이름** | AgentDesk |
| **정의** | AI 에이전트 오피스 시뮬레이터 — CEO 데스크에서 AI 에이전트 팀을 지휘 |

| 항목 | 내용 |

| **현황** | **구현 완료.** `GET /api/task-reports/:taskId/artifacts/download` 다운로드 엔드포인트(zip 다운로드) 존재. `ArtifactList.tsx`에서 다운로드 버튼(`getTaskArtifactDownloadUrl`) 연동 완료. |

| 항목 | 내용 |
|------|------|
| **현황** | Telegram·Discord는 Receiver(수신) 구현 완료. Signal·iMessage·WhatsApp·Google Chat은 **발신(Send)만** 구현, **수신(Receiver/Webhook)** 미구현. |
| **필요 기능** | • Signal: signal-cli REST API 기반 수신 폴링 또는 웹훅<br>• WhatsApp: Meta Webhook 수신 엔드포인트<br>• Google Chat: Pub/Sub 또는 Bot API 수신<br>• iMessage: macOS 전용, 수신 구현 난이도 높음 (우선순위 낮음) |

| **자동 트래킹** | 게이트 진행 상태·리드타임·SLA 위반을 CSV/모니터링 로그로 기록 |

| 항목 | 내용 |
|------|------|
| **무결성 검증** | 배포 산출물 SHA-256 기준선·버전 고정, 배포 전 재검증 |
| **RBAC** | PPT/협업 저장소 권한 최소화 점검(icacls), 기준 충족 시 배포 허용 |
| **CI 보안 게이트** | 비밀정보 0건, 악성 매크로/첨부 검사, 접근권한 변경 이력 제출 |
