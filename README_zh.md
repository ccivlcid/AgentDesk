# AgentDesk

AI代理办公室模拟器 - 从CEO桌面指挥你的AI代理团队。

## 主要功能

- 像素艺术动画办公室
- CEO仪表板（KPI指标、代理排名）
- 多AI提供商支持（Claude Code、Codex CLI、Gemini CLI等）
- 看板任务管理
- 消息集成（Telegram、Discord、Slack等）
- 多语言UI（英语、韩语、日语、中文）

## 要求

- Node.js >= 22
- pnpm >= 10

## 快速开始

```bash
pnpm install
cp .env.example .env
pnpm setup
pnpm dev
```

在浏览器中打开 http://localhost:8800。
