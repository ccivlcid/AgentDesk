# AgentDesk

AIエージェントオフィスシミュレーター - CEOデスクからAIエージェントチームを指揮しましょう。

## 主な機能

- ピクセルアートアニメーションオフィス
- CEOダッシュボード（KPIメトリクス、エージェントランキング）
- 複数AIプロバイダー対応（Claude Code、Codex CLI、Gemini CLI等）
- カンバンボードによるタスク管理
- メッセンジャー連携（Telegram、Discord、Slack等）
- 多言語UI（英語、韓国語、日本語、中国語）

## 要件

- Node.js >= 22
- pnpm >= 10

## クイックスタート

```bash
pnpm install
cp .env.example .env
pnpm setup
pnpm dev
```

ブラウザで http://localhost:8800 を開いてください。
